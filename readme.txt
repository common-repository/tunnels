== Description ==

= Tunnels =
The main function of this is to publish your wordpress post to social medias, you'll need a API key to enable this plugin.

**More info:**

* [tunnels](http://www.tunnels.me/)
* Read more great [WordPress tips](http://www.wp-coder.net/blog)

== Installation ==

1. Upload the contents of tunnels.zip to your plugins directory.
2. Activate the plugin
3. There will be a "Tunnels" menu on your admin panel, you need to sigup on tunnels.me to get a API key and verify it. 
4. SignIn your twitter and facebook account on http://tunnels.me.  
5. That's all. There will be a post on twitter and facebook when you publish a new post!
